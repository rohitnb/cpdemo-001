## How to run this?

1. Create the virtual env

```bash
python3 -m venv .venv
```

2. Activate the virtual env

```bash
source .venv/bin/activate
```

3. Install the dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
flask run
```
